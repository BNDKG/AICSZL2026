# 已退役的特征探索方向

## 2026-07：价格成交量扩展与分散化扩展

本项目曾依次探索过两批扩展特征：第一批是在 `base_v1` 之外加入5个价格、反转、波动率、成交额放大和区间位置特征；第二批再加入10个估值、规模、换手、流动性、资金流、日内强弱和回撤特征。

在相同训练、预测和回测口径下，扩展没有带来增量价值。已完成的5特征对10特征实验中，base5 净收益约为 `-0.00416`，扩展后约为 `-0.1496`，固定种子随机 baseline 约为 `-0.1099`；后续20特征组合也弱于当时的10特征基线。因此这些实现、测试、特征组和可执行配置已全部删除，不属于当前系统能力。

后续不要原样重新加入以下方向的同构组合：

- 单纯增加20日收益、1日反转、20日波动、5日成交额比和20日价格区间位置排名。
- 把常见估值、规模、换手、成交量比、Amihud、大小单资金流、日内强弱和20日回撤排名一次性打包。

这不表示任何单因子永久无效，而是表示“按当时定义直接打包并使用相同模型”的路线已被实证否定。再次探索必须先提出不同的经济假设、正交性或稳健性证据，并作为小型独立实验组验证；不得仅换名恢复已删除的组合。
